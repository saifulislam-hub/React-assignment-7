# 🌐 KinKeeper

## 📌 Project Description
KinKeeper is a relationship management web application built with React. It helps users keep track of their friends, manage communication frequency, and maintain meaningful connections through reminders and timeline tracking.

---

## 🚀 Technologies Used
- React.js
- React Router
- Tailwind CSS
- JavaScript (ES6+)
- JSON (Mock Data)

---

## ✨ Key Features

### 1. 👥 Friend Management
- View a list of friends with detailed profiles
- Includes tags, status, and contact preferences

### 2. ⏰ Relationship Tracking
- Track days since last contact
- Set relationship goals (e.g., connect every 30 days)

### 3. 📊 Timeline & Quick Actions
- Add interaction history (call, text, video)
- Quick check-in buttons for fast updates
